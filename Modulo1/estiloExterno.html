Componentes/etiquetas llamado css

  • Etiquetas como "p" "h2" -> se llaman por su nombre
  • Clases -> se llaman como vos quieras, pero deben tener un "PUNTO" adelante
    si la llamo "parrafoPrincipal" cuando la trabajo en el css, la llamo .parrafoPrincipal
  • Ids (identificadores) NO se repiten en la misma archivo html. Lo llamas como vos queres y cuando trabajo en el css, le coloco # adelante.
    id llamdo destacado, lo uso dentro del css como #destacado{estilos que quiera}
